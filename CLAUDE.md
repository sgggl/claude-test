# CLAUDE.md

이 파일은 저장소에서 작업할 때 Claude Code(claude.ai/code)에게 제공되는 안내 문서입니다.

## 프로젝트 개요

유명 명언을 보여주는 단일 파일 정적 웹앱(`index.html`)입니다. 빌드 도구, 외부 의존성, 서버가 필요 없으며 브라우저에서 파일을 바로 열어 사용합니다.

## 구조

모든 코드는 `index.html` 내 세 개의 인라인 섹션으로 구성됩니다.

- **`<style>`** — 전체 CSS. `rgba` 레이어링을 활용한 다크 테마. 카테고리별 색상 테마는 `.tag-{카테고리}` 패턴을 따릅니다 (예: `.tag-인생`, `.tag-성공`).
- **`<body>`** — 시맨틱 HTML 구조: 헤더 → 히어로 섹션 → 필터 바 → 검색 입력 → 카드 그리드 → 푸터.
- **`<script>`** — 프레임워크 없는 순수 JS. 주요 데이터 및 함수:
  - `quotes[]` — `{ text, author, category }` 객체로 구성된 명언 마스터 배열
  - `randomHero()` — 페이드 트랜지션과 함께 히어로 패널에 새 랜덤 명언 표시
  - `buildFilter()` — `quotes[]`를 기반으로 카테고리 필터 버튼을 동적 생성
  - `renderGrid()` — 활성 카테고리와 검색어로 `quotes[]`를 필터링해 카드 그리드를 다시 렌더링

## 명언 추가 방법

`<script>` 섹션의 `quotes` 배열에 객체를 추가합니다.

```js
{ text: "명언 내용", author: "작가 이름", category: "카테고리" }
```

사용 가능한 카테고리(각각 CSS 색상 테마 보유): `인생`, `성공`, `지혜`, `사랑`, `용기`, `행복`.  
새 카테고리를 추가할 경우, 기존 패턴을 참고하여 `.tag-{새카테고리}` CSS 규칙도 함께 추가해야 합니다.
